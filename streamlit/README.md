# Streamlit Demos

[Streamlit](https://streamlit.io/) is an amazing library to rapidly develop data apps!

## Install Streamlit

```bash
	$  pip install streamlit
```

## Run

```bash
	$  streamlit run <file.py>
```

## Demos

### Uber Pickups in NYC

```bash
	$  streamlit  run uber.py
```

Go to URL printed out, usually http://localhost:8501
